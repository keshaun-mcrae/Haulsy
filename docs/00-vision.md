# Haulsy — Vision

Haulsy is a mobile-first Surrey marketplace for furniture/electronics/appliances + cars with clean UI, chat translation-on-demand, and future AR fit-check.

## Target users
- Local sellers who want to post fast and sell safely
- Local buyers who want clean browsing and fast scheduling

## Why it wins
- Familiar marketplace UX (FB) but cleaner (Apple)
- Chat quick actions + scheduling + translation toggle
- Safety-first flows (report/block + meetup guidance)

## 30-day success metrics
- Listings/week
- % listings that receive a message
- % chats that schedule a meetup
- Median time to first message
