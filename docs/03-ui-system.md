# UI System

Vibe: Facebook Marketplace familiarity + Apple clean minimal + subtle Uber patterns (chips/sheets/sticky bars).

## Rules
- White/near-white background
- 1 font, max 2 weights (regular/semibold)
- Rounded corners 12–16px
- Soft borders, minimal shadows
- Max 2 primary actions per screen

## Core components
- TopBar (location pill + search + filter)
- CategoryChips (horizontal)
- ListingCard + ListingGrid (2-column)
- FilterSheet (bottom sheet)
- StickyActionBar (listing detail)
- ChatHeader with Translate toggle + language pill
