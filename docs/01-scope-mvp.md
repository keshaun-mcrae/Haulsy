# MVP Scope

## IN (MVP)
- Auth
- Browse feed (grid) + search + basic filters
- Listing detail + message seller
- Sell wizard (3 steps: Photos ? Details ? Publish)
- Inbox list + chat thread
- Mark sold / archive
- Report + block

## OUT (Not MVP)
- Payments/escrow
- Ratings/reviews
- Full disputes
- AR (ships later)
- Full delivery marketplace (delivery is v0-lite later)
